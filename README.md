# Modal Fretboard

**Guitar fretboard visualizer highlighting the signature notes that define each mode's signature sound.**

🎸 **[Launch App](https://lightisbeauty.github.io/Modal/)**

---

## What it does

Modal Fretboard shows you where every note of a mode lives on the guitar neck, color-coded so you can immediately see what makes each mode sound the way it does.

| Color | Meaning |
|---|---|
| 🟠 Orange | Root |
| 🟡 Yellow | Butter notes — the 3rd, plus the intervals that define this mode's character |
| 🔵 Blue | Remaining scale notes |

## Views

- **Box** — 5-fret position window rooted on the low E string
- **15 frets** — full neck overview
- **24 frets** — extended range

## Modes

All 7 modes of the major scale, in all 12 keys.

| Mode | Character |
|---|---|
| Ionian | maj3 + maj7 — the resolved major sound |
| Dorian | b3 + natural 6 — minor with a raised, hopeful 6th |
| Phrygian | b2 + b6 — the flat 2nd is what makes it Phrygian |
| Lydian | #4 + maj3 + maj7 — the raised 4th gives it a floating quality |
| Mixolydian | maj3 + b7 — dominant tension without resolving |
| Aeolian | b3 + b6 — the flat 6th separates it from Dorian |
| Locrian | b2 + b5 — the diminished 5th makes it inherently unstable |

## Tech

Single self-contained HTML file. No dependencies, no build step. Open in any browser.
